# ClinicaVoice - MUI Themed Frontend (Mock API)

This project is a Material UI rework of the ClinicaVoice frontend with a mock API.

How to run:
1. unzip the project
2. cd ClinicaVoice_MUI_Themed
3. npm install
4. npm run dev

Note: This archive does not include node_modules. Install dependencies with npm install.
